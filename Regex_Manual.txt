TvAIr 正規表現マニュアル

1. このファイルについて

このファイルは、TvAIr の自動検索予約などで正規表現を使うための簡易マニュアルです。

通常の番組名検索では、番組名の一部をそのまま入力すれば検索できます。
表記ゆれをまとめて検索したい場合や、複数の条件をまとめたい場合に正規表現を使います。

2. 基本

文字をそのまま書くと、その文字を含む番組を検索します。

例:
ニュース

「ニュース」を含む番組に一致します。

3. 表記ゆれをまとめる

一部の文字が違う番組名をまとめたい場合は、角括弧 [] を使います。

例:
ドラマ[新終]

次のような表記に一致します。

- ドラマ新
- ドラマ終

4. [字]、[新]、[終] などを扱う

番組名に [字]、[新]、[終] などの表記が付く場合があります。

角括弧そのものを検索したい場合は、\ を付けます。

例:
\[新\]ドラマ

「[新]ドラマ」に一致します。

例:
ドラマ\[終\]

「ドラマ[終]」に一致します。

5. どちらかに一致させる

複数の候補のどちらかに一致させたい場合は、| を使います。

例:
ニュース|報道

「ニュース」または「報道」に一致します。

6. グループ化する

| の範囲をまとめたい場合は、丸括弧 () を使います。

例:
(ニュース|報道)ステーション

次のような表記に一致します。

- ニュースステーション
- 報道ステーション

7. 話数に一致させる

数字に一致させたい場合は \d を使います。

例:
第\d話

次のような表記に一致します。

- 第1話
- 第2話
- 第9話

複数桁にも一致させたい場合は + を使います。

例:
第\d+話

次のような表記に一致します。

- 第1話
- 第12話
- 第100話

8. 任意の文字に一致させる

任意の文字列に一致させたい場合は .* を使います。

例:
ドラマ.*最終回

「ドラマ」と「最終回」の間に任意の文字が入っても一致します。

9. 特殊文字をそのまま検索する

次の文字は正規表現で特別な意味を持ちます。

. ^ $ * + ? { } [ ] \ | ( )

これらを文字として検索したい場合は、前に \ を付けます。

例:
映画\.com

「映画.com」に一致します。

例:
A\+B

「A+B」に一致します。

10. 広すぎる条件に注意

広すぎる正規表現は、意図しない番組まで一致することがあります。

例:
.*

すべての文字列に一致するため、通常は使わないでください。

例:
ニュース.*

「ニュース」で始まる多くの番組に一致します。
必要に応じて条件を絞ってください。

11. よく使う例

例:
ニュース|報道|天気

ニュース、報道、天気のいずれかに一致します。

例:
(アニメ|映画).*\[新\]

アニメまたは映画を含み、さらに [新] を含む番組に一致します。

例:
第\d+話|最終回

「第1話」「第12話」などの話数、または「最終回」に一致します。

12. 注意

正規表現は便利ですが、条件が広すぎると不要な予約が増える場合があります。
最初は短い条件で確認し、必要に応じて少しずつ絞り込んでください。
